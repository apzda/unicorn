# 单体应用

单体应用启动器

## 已集成的服务

| # | 名称      | 说明                                                        |
|:--|:--------|:----------------------------------------------------------|
| 1 | config  | 应用设置(首选项)服务                                               |
| 2 | uc      | 用户中心，提供用户管理相关功能                                           |
| 3 | audit   | 审计服务                                                      |
| 4 | captcha | 验证码服务                                                     |
| 5 | oss     | 对象存储服务, 后端已支持: 阿里云OSS、腾讯云COS、MinIO和本地存储。内置插件: 添加水印和Resize |

## 可选服务

| # | 名称        | 说明                  |
|:--|:----------|:--------------------|
| 1 | sms       | 短信服务, 通道支持: 阿里大鱼、创蓝 |
| 2 | messenger | 信使服务                |
| 3 | wallet    | 用户钱包                |
